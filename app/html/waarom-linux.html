Waarom Linux gebruiken

Is het geen tijd om over te schakelen naar Linux? 15 redenen om Windows te verlaten

BOTTOM  |  TOP

  1. Linux is vrij.

  2. Linux is gratis.

  3. Linux is stabiel.

  4. Geen virussen.

  5. Geen serienummers

  6. Geen kwaadaardige software (malware).

  7. Goede taalondersteuning, 99.9% in het Nederlands.

  8. Start op vanaf de Ubuntu Live-CD of Live-USB en controleer of alles werkt.

  9. Makkelijk installeren in slechts enkele stappen, met name bij Ubuntu Linux.

  10. Linux draait ook op oudere hardware, niet nodig om een nieuwe computer te kopen.

  11. Updates uitvoeren geldt voor het hele systeem; het besturingssysteem en alle software.

  12. Installeer software met één muisklik, een keuze uit meer dan 40.000 pakketten, voor elk wat wils.

  13. Veel, heel veel, hulp op het Internet, bijvoorbeeld via Ubuntu en Ubuntu Documentatie in het Nederlands, en Ask Ubuntu in het Engels.

  14. Diverse bureaubladomgevingen beschikbaar zoals GNOME in Ubuntu, MATE in Ubuntu MATE, KDE in Kubuntu, Xfce in Xubuntu, of LXDE in Lubuntu.

  15. U kunt werken zoals u gewend bent met Office ( LibreOffice), IM (instant Messaging) ( Pidgin), Web ( Firefox), en Mail ( Thunderbird), zie ook Open Source voor Windows en Open Source voor Mac.
