Why use Linux

Is het geen tijd om over te stappen naar Linux? 16 redenen om Windows te verlaten
Isn't it time to switch to Linux? 16 reasons to leave Windows

BOTTOM  |  TOP

  1. Linux is vrij.

  2. Linux is gratis.

  3. Linux is stabiel.

  4. Geen virussen.

  5. Geen serienummers.

  6. Geen kwaadaardige software (malware).

  7. Er zijn bedrijven die het gebruik van Windows om bovenstaande redenen niet toestaan, zie The Register.

  8. Goede taalondersteuning, 99.9% in het Nederlands.

  9. Start op vanaf de Ubuntu of Debian Live usb-stick (of dvd) en controleer eerst of alles werkt alvorens te installeren.

  10. Makkelijk installeren in slechts enkele stappen, met name bij Ubuntu en Debian.

  11. Linux draait ook op oudere hardware, niet nodig om (steeds) een nieuwe computer te kopen.

  12. Updates uitvoeren geldt voor het hele systeem; het hele besturingssysteem en alle software.

  13. Installeer software met één muisklik, een keuze uit meer dan 58.000 pakketten, voor elk wat wils.

  14. Veel, heel veel, hulp op het Internet, bijvoorbeeld via Ubuntu Nederland en Ubuntu Documentatie in het Nederlands, en Ask Ubuntu in het Engels.

  15. Diverse bureaubladomgevingen beschikbaar zoals GNOME in Ubuntu, KDE in Kubuntu, Xfce in Xubuntu, MATE in Ubuntu MATE, of LXDE in Lubuntu.

  16. U kunt werken zoals u gewend bent met Office (LibreOffice), IM (instant Messaging) (Pidgin), Web (Firefox), en Mail (Thunderbird), zie ook Open Source voor Windows en Open Source voor Mac.

  1. Linux is free.

  2. Linux is gratis.

  3. Linux is stable.

  4. No virusses.

  5. No serial numbers.

  6. No malicious software (malware).

  7. There are companies that do not allow the use of Windows for the above reasons, see The Register.

  8. Good language support.

  9. Boot from the Ubuntu or Debian Live USB stick (or DVD) and check that everything works before installing.

  10. Easy installation in just a few steps, especially with Ubuntu and Debian.

  11. Linux also runs on older hardware, no need to (always) buy a new computer.

  12. Performing updates is system-wide; the entire operating system and all software.

  13. Install software with one click, from over 58,000 packages, something for everyone.

  14. Lots, lots of help on the Internet, for example via Ask Ubuntu.

  15. Various desktop environments available such as GNOME in Ubuntu, KDE in Kubuntu, Xfce in Xubuntu, MATE in Ubuntu MATE, or LXDE in Lubuntu.

  16. You can work as usual with Office (LibreOffice), IM (instant Messaging) (Pidgin), Web (Firefox), and Mail (Thunderbird), see also Open Source for Windows and Open Source for Mac.
